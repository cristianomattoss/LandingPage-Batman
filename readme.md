# 🦇 Landing Page Batman

Uma landing page inspirada no universo do Batman, desenvolvida para praticar HTML, CSS e JavaScript.

## 🚀 Tecnologias utilizadas
* HTML5
* CSS3
* JavaScript

## ✨ Funcionalidades
* Layout responsivo
* Alternância de tema (Dark/Light)
* Interface inspirada no universo Batman
* Organização modular de arquivos

## 📂 Estrutura do projeto
```bash
LandingPage-Batman/
│── index.html
│── CSS/
│   └── style.css
│── JS/
│   └── script.js
│── IMG/
│── README.md
```
